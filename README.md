# 💡 Hardware Made Easy Workshop  
**Girls Who Code College Loops @ Stan State**

Welcome! This repository contains the code and materials for the **Hardware Made Easy Workshop**, where we explore the basics of working with an **Arduino + LCD display**.

This workshop is designed for beginners and walks through progressively more interactive and creative LCD projects.

---

## 📚 Workshop Overview

### 🟢 Lesson 1: Hello World on an Arduino LCD
- File: `hello.ino`
- Learn how to:
  - Set up your Arduino LCD
  - Display your first message  
- Classic starting point: printing “Hello, World!”

---

### 🟡 Lesson 2: Displaying and Cycling Through Text
- File: `quote.ino`
- Learn how to:
  - Display multiple messages
  - Cycle through text automatically  
- Great for showing your favorite quotes or dynamic information

---

### 🔵 Lesson 3: Animating Text on an Arduino LCD
- File: `personalized_badge.ino`
- Learn how to:
  - Create moving/scrolling text
  - Build a personalized digital name badge  
- Adds motion and personality to your display

---

### 🟣 Lesson 4: Custom Characters & Animation
- File: `emoji_animation.ino`
- Learn how to:
  - Design custom LCD characters (like emojis!)
  - Animate them on screen  
- This is where things get creative 🎨

---

## 🛠️ Requirements

- Arduino board (Uno, Nano, etc.)
- 16x2 LCD display (with or without I2C module)
- Arduino IDE installed
- Basic wiring setup (provided during workshop)
